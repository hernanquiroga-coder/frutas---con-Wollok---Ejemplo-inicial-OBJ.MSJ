# Frutas Conmigo

Proyecto sencillo en HTML para mostrar una lista de frutas y practicar el uso de objetos y DOM.

## Estructura del proyecto

- `index.html`: página principal del proyecto.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Interactúa con la página para ver los datos de las frutas.

## Objetivo

Este proyecto está diseñado para practicar:

- HTML básico
- JavaScript para manipular objetos y el DOM
- Crear contenido interactivo en el navegador

## Autor

- Proyecto creado como ejemplo educativo.
